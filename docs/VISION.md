## ğŸ’° Referral Program

### Overview
A commission-based referral system where individuals can earn money by bringing vendors onto the platform.

### How It Works
1. **Sign Up:** Anyone can create a referral account
2. **Get Code:** Receive a unique referral code
3. **Recruit Vendors:** Share code with potential vendors
4. **Earn Commissions:** Get paid when they sign up and sell

### Commission Structure

| Commission Type | Description | Payout Frequency |
|----------------|-------------|------------------|
| **Registration Commission** | One-time bonus when referred vendor signs up | Weekly |
| **Sales Commission** | Percentage of platform engagement fees | Weekly |
| **Passive Income** | Recurring commission on all sales from referred vendors | Weekly (Ongoing) |

### Payment Schedule
- **Payout Day:** Weekly
- **Minimum Threshold:** TBD
- **Payment Methods:** Mobile Money, Bank Transfer
